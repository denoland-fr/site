# Website

Ce site est lancé avec l'aide de [Docusaurus 2](https://v2.docusaurus.io/) et Netlify.

### Installation

```
$ yarn
```

### Local Dev

```
$ yarn start
```

Commande utilisé pour lancer en local le site de Denoland FR.

### Build

```
$ yarn build
```

Commande utilisé principalement pour le déploiement.
