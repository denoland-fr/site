## Installation

Avec Shell (macOS, Linux):
```
curl -fsSL https://deno.land/x/install/install.sh | sh
```

Avec Homebrew (macOS):
```
brew install deno
```

Avec Chocolatey ou Scoop (Windows):
```
choco install deno
# OR
scoop install deno
```
